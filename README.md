## Como Rodar
### Clone o repositório
```bash
$ git clone https://github.com/MarlonVictor/uiClone.git
```

### Executar a aplicação
```bash
# Instalar Dependências
$ yarn install

# Executar Aplicação 
$ yarn start
```
* Vá para [http://localhost:3000](http://localhost:3000/) para ver o resultado.
